  1. As of now, the software will accept only one sheet.
  2. File should have xlsx extension.
  3. If you have created multiple table in one sheet, it will only accept one table at a time.
  4. Remove unnecessary details. (Links, Images, etc. )
  5. Each cell should contain only one value. (Ex: mobile number.)
  6. Column headers are fetched from row 1 of the xlsx file.
